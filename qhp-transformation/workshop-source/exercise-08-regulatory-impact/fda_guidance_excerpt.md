# FDA Guidance Excerpt — Oncology Trial Endpoint Reliability

Key themes from the March 2026 draft guidance:
- Progression-free survival endpoints should include clear adjudication procedures when imaging timing varies across sites.
- Patient-reported outcome instruments used in labeling discussions must demonstrate consistent administration windows and low missingness.
- Sponsors should pre-specify handling of informative missing data where treatment discontinuation differs across arms.
- Subgroup analyses discussed externally should match the hierarchy described in the statistical analysis plan.
