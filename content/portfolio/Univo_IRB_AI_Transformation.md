# Univo IRB: AI Transformation Strategy

## Executive Summary

**Univo IRB** is QHP's purpose-built Independent Review Board serving biotechnology and small pharmaceutical companies. With strong customer service, operational flexibility, and rapid turnaround as core differentiators, Univo is positioned to become the **first AI-native IRB**—transforming how ethical review is conducted while maintaining uncompromising human oversight. This strategy could achieve **40-60% reduction in review turnaround times**, **30% operational cost savings**, and establish Univo as the technology leader in an industry dominated by legacy players.

## Product & Services Layer: AI-Enhanced Offerings

### Priority 1: AI-Assisted Protocol Review Engine 🔥

**Current State:** Human reviewers analyze protocols manually
**Target State:** AI pre-analysis with human expert oversight

**Critical Principle:** IRB review requires human judgment—AI assists, never replaces. The goal is to make reviewers more effective, not to automate decisions.

**Technical Approach:**

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PROTOCOL SUBMISSION                              │
│              (via OneVerse client portal)                           │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    AI PROTOCOL ANALYSIS ENGINE                      │
│                                                                     │
│  DOCUMENT PROCESSING:                                               │
│  • Extract study design, objectives, endpoints                      │
│  • Identify patient population, I/E criteria                        │
│  • Map risk categories (minimal, greater than minimal)              │
│  • Detect novel elements requiring special attention                │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    AI COMPARISON & FLAGGING                         │
│                                                                     │
│  • Compare to similar approved protocols                            │
│  • Flag regulatory compliance concerns                              │
│  • Highlight informed consent alignment issues                      │
│  • Identify questions for investigator                              │
│  • Generate structured summary for reviewer                         │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    REVIEWER WORKBENCH                               │
│                                                                     │
│  AI-GENERATED OUTPUTS:                                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐                 │
│  │ Structured  │  │   Risk      │  │  Suggested  │                 │
│  │ Protocol    │  │ Assessment  │  │  Questions  │                 │
│  │ Summary     │  │ Matrix      │  │ for PI      │                 │
│  └─────────────┘  └─────────────┘  └─────────────┘                 │
│                                                                     │
│  HUMAN REVIEWER ACTIONS:                                            │
│  • Verify AI analysis accuracy                                      │
│  • Apply ethical judgment                                           │
│  • Make determination (approve/modify/defer)                        │
│  • Document rationale                                               │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    COMMITTEE PRESENTATION                           │
│                                                                     │
│  • AI-prepared presentation materials                               │
│  • Key discussion points highlighted                                │
│  • Historical context (similar protocols)                           │
│  • Human deliberation and decision                                  │
└─────────────────────────────────────────────────────────────────────┘
```

**AI Analysis Components:**

| Component | Technology | Output |
|-----------|------------|--------|
| **Protocol Parser** | Claude + structured extraction | JSON representation of protocol elements |
| **Risk Classifier** | ML on historical reviews | Risk category prediction |
| **Consent Analyzer** | NLP comparison | ICF-protocol alignment report |
| **Regulatory Checker** | Rules + AI | Compliance flag list |
| **Similar Protocol Finder** | Embeddings + vector search | 5 most similar approved protocols |

**Key Safeguards:**

| Safeguard | Implementation |
|-----------|----------------|
| **Human Final Decision** | All determinations made by qualified reviewers/committee |
| **AI Transparency** | Clear labeling of AI-generated content |
| **Audit Trail** | Complete logging of AI analysis + human review |
| **Escalation Paths** | Complex cases flagged for senior review |
| **Bias Monitoring** | Regular audits of AI recommendations |

**Investment:** $200-350K development
**Timeline:** 9-12 months
**ROI:** 40-50% reduction in review time → higher throughput, same quality

**Regulatory Positioning:** Position as "AI-augmented human review" to FDA and accreditation bodies.

**Target State:** AI-powered change detection and impact assessment

**Technical Approach:**

| Capability | Technology | Value |
|------------|------------|-------|

| **Change Classification** | ML classification | Minor/administrative vs. substantive |

| **Historical Context** | RAG on past amendments | Similar amendment handling |
| **Reviewer Recommendation** | AI suggestion | Expedited vs. convened review path |

**Sample Amendment Analysis:**

```
AMENDMENT #4 ANALYSIS
=====================

CHANGES DETECTED: 7

SUBSTANTIVE CHANGES (3):
1. Added Arm C (new treatment group)
   - Impact: Requires consent update, risk assessment
   - Recommendation: Convened review required
   
2. Modified inclusion criterion #5 (age range 18-65 → 18-75)
   - Impact: Expanded population, potential age-related risks
   - Recommendation: Committee discussion advised

3. Added MRI at Visit 8
   - Impact: New procedure, contraindication screening needed
   - Recommendation: Consent update required

ADMINISTRATIVE CHANGES (4):
1. Sponsor address update
2. Typo corrections (3 instances)
3. Reference number formatting
4. Version date update

RECOMMENDED PATH: Convened Board Review
ESTIMATED REVIEW TIME: 45 minutes (vs. 90 min traditional)
```

**Investment:** $100-150K development
**Timeline:** 6-9 months
**ROI:** 50% faster amendment processing, better change tracking
